# Zavi Audiobook

## Tech stack
- HTML
- CSS
- JavaScript

## How run
1. Clone repositori ini:
   `git clone https://github.com/khalishalya/zavi-audiobook.git`
2. Masuk ke folder proyek:
   `cd zavi-audiobook`
3. Buka file `index.html` di browser web (Chrome, Firefox, Edge) atau gunakan ekstensi Live Server di editor kodemu.
